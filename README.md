# telegram-signup

Sign up for Telegram with any device.

## Requirements

- `node` v7.6.0+
- `npm`

## Installation

Via `npm`:

    npm install -g telegram-signup

## Usage

Start from command line:

    telegram-signup

It will request phone number, first name, and last name (optional).
